# BABADPROJECT 
## badminton booking db system (ฐานข้อมูลบริหารจัดการสนามแบดมินตัน)

## 1.ผู้จัดทำ
 	นายนรวิชญ์ มากปรางค์ 67102010164
  นางสาววริศรา ดิลกกาญจนมาลย์ 67102010173
  นางสาวนิวีฟ้าค แวดือเระ 67102010518

## 2.รายละเอียด
 	ระบบฐานข้อมูลสำหรับบริหารจัดการการจองสนามแบดมินตัน
  ช่วยให้เจ้าของสนามสามารถจัดการข้อมูลสาขา สนาม ลูกค้า และรายการจองได้อย่างเป็นระบบ

  รองรับการบันทึกตารางเวลาใช้งาน คำนวณราคารวม และตรวจสอบข้อมูลย้อนหลังได้
  ออกแบบฐานข้อมูลตามหลัก Normalization ถึงระดับ 3NF
  และพัฒนา/ทดสอบด้วย PostgreSQL
  
## 3.โครงสร้างฐานข้อมูล
	Tables หลักของระบบ
  1. Owner (ข้อมูลเจ้าของสนาม)
  2. Branch (ข้อมูลสาขา)
  3. Court (ข้อมูลสนาม)
  4. Customer (ข้อมูลลูกค้า)
  5. Booking (ข้อมูลการจอง)
     
## 4.วิธีใช้งานโปรเจค
	1. เปิด PostgreSQL
  2. สร้าง Database ใหม่ใน PostgreSQL
  3. รันไฟล์ SQL Implementation
  4. ทดลอง Query
